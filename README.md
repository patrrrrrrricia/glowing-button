## 💻 Overview
**GlowingButton** is a bespoke digital component designed to harmonize high-fashion aesthetics with interactive web technology. It serves as an immersive visual signature, transforming a standard UI element into a glowing centerpiece. Built on a clean **HTML5/CSS3** core, it utilizes advanced layering techniques to simulate real neon luminescence.

---

## 🤍 Tech Stack
* **Core:** HTML5, CSS3 (Custom Properties & Keyframe Animations)
* **Styling:** Neon Glow Layering, Box-Shadow Stacking
* **Design:** Responsive Web Design (RWD), Minimalism

## 🎀 Core Functionality
* **Luminescence Engine:** Multi-layered `box-shadow` and `text-shadow` configurations to create a realistic light bleed effect.
* **Interactive Hover:** Dynamic scaling and intensity modulation upon user interaction.
* **Fluid Animation:** Pulse-effect keyframes that simulate the "breath" of a real neon sign.
* **Zero-Dependency Architecture:** Pure CSS implementation ensuring ultra-fast loading and 100% performance efficiency.

---

## 📂 Project Structure
The project follows a minimalist modular architecture for static deployment:
* **`index.html`**: The structural backbone of the digital signature.
* **`style.css`**: The "Couture" layer containing the neon algorithms and visual logic.
* **`photo/`**: Centralized repository for high-resolution visual documentation.
* **`GitHub Pages`**: Automated CI/CD pipeline for real-time web hosting.
